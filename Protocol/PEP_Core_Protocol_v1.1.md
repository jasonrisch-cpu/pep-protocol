2. Definitions

2.1 Cost (C)

C is the true unit cost of producing the exchangeable output.

C must be honest, not inflated and not minimized.

C may include:

materials

consumables

tools (amortized)

energy

space

system maintenance

time (mode-dependent)


Hiding costs from yourself = self-extraction.


---

2.2 Operating Modes (Critical)

PEP operates in three explicit modes.

Mode A ‚Äî Unit Cost (Product-Based)

Use when:

work is side-income or supplemental


C includes: materials, tools, energy
C excludes: living expenses, time

Examples: crafts, patches, garments, goods


---

Mode B ‚Äî Hourly Cost (Time-Based)

Use when:

C includes: life maintenance prorated by billable hours

Formula example:
Monthly costs √∑ billable hours = hourly C

Examples: consulting, caregiving, coaching


---

Mode C ‚Äî Hybrid (Job-Based)

Use when:

Examples: trucking, contracting, repair, installation


---

Mode selection is a calibration choice.
Mode-switching is allowed if it reduces distortion.

Mode-switching to avoid discomfort = misuse.
